## What is this?
Imagine if you could hire a **digital company** instantly. 
You are the CEO. You have an idea. Instead of hiring 10 people, finding an office, and managing them effectively, you just type your idea into a "Magic Window," and this system does the rest.

**Anti-Gravity** is that system. It is a "Multi-Agent AI" platform. That's a fancy way of saying it's a team of specialized AI bots that work together like employees in a company.

## The "Employees" (The Agents) üëî
We didn't just build one smart chatbot (like ChatGPT). We built a **team**.

### 1. The CIO (The Boss) - *The Orchestrator*
*   **Role**: Coordinates everything.
*   **Job**: When you ask for something ("Build me a game"), the CIO doesn't write code. The CIO thinks: *"Okay, to build a game, I need a plan, some research, and a coder."* Then it assigns tasks to the others.

### 2. The Specialist - *The Planner*
*   **Role**: The Architect.
*   **Job**: Takes a big, scary goal and breaks it down into small, easy steps. It creates the roadmap.

### 3. The Researcher - *The Librarian*
*   **Role**: Facts & Information.
*   **Job**: Uses Google to find real answers. If you ask about stock prices or new software libraries, this agent finds the truth so the others don't guess.

### 4. The Coder - *The Engineer*
*   **Role**: Builder.
*   **Job**: Writes the actual software code. It can fix its own errors and run tests.

### 5. The Critic - *The Safety Inspector*
*   **Role**: Quality Control.
*   **Job**: Reviews everything the Coder and Researcher do. If the code is buggy or unsafe, the Critic rejects it and sends it back. Nothing gets to you without approval.

### 6. The Memory - *The File Cabinet*
*   **Role**: Long-term Storage.
*   **Job**: Remembers what you did yesterday, last week, or last month. It ensures the system gets smarter over time.
